---
title: "Metadados da tabela"
openapi: "GET /v2/workspace/{workspaceId}/tables/{tableId}/metadata"
---

Retorna os metadados de uma tabela: suas colunas (nome, tipo, posição, opções, configurações) e o total de linhas. Use esse endpoint para entender a estrutura da tabela antes de consultar ou manipular seus dados.
